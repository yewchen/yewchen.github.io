SMPP 是什麼

發表於 2018-08-15 更新於 2020-06-10 分類於 協定相關 Disqus：

前言

  1. 國內與SMPP相關的文章整理分享很少，硬嗑文獻又很痛苦。雖然網路上找的到翻譯過的文獻，但其實閱讀門檻依舊很高，所以自己邊研讀邊整理。目前尚未有實作的整理筆記，未來會一一納入。
  2. 研讀的版本為《Short Message Peer to Peer Protocol Specification v3.4》。
  3. SMPPv5.0主要是增加對CBS(Cell Broadcast Centres)的interface，以及增加一些TLV的實作，可以參考這篇文章。
  4. SMSC會進行ss7信號網路交換，因此許多SMPP的操作背景知識也與ss7相關。可以去參考SMPPv3.4中的1.4Reference：GSM 03.40 v5.7.1、GSM 03.38 v5.5.1、GSM MAP 09.02 v5.11.0。
  5. 研讀中，尚未有機會實際碰觸了解，僅是紀錄了一些目前覺得相對重要的東西（偏實作技術面）。
